## 1. 系统简介

Stock Analyzer 是一款智能股票分析系统，能够自动获取股票数据、计算技术指标、生成分析报告。

### 核心功能

- ✅ **自动数据获取**: 从 Tushare 获取实时股票行情
- ✅ **技术指标计算**: EMA、RSI、MACD、ATR、布林带等
- ✅ **趋势分析**: 多维度评分系统
- ✅ **报告生成**: HTML 格式分析报告

### 分析维度

| 维度 | 说明 | 评分范围 |
|------|------|----------|
| 趋势评分 | 基于均线排列判断趋势方向 | 1-5 |
| 动量评分 | 基于RSI/MACD判断买卖信号 | 1-5 |
| 成交量评分 | 基于量能变化判断活跃度 | 1-5 |
| **综合评分** | 加权综合评分 | 1-5 |

### 投资建议等级

| 评分 | 建议 | 说明 |
|------|------|------|
| ≥4.0 | 买入 | 强烈看多信号 |
| 3.5-4.0 | 持有偏多 | 有上涨潜力 |
| 2.5-3.5 | 持有 | 观望为主 |
| <2.5 | 卖出 | 风险较大 |

## 2. 环境准备

### 2.1 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Linux / macOS / Windows |
| Python | 3.11+ |
| 内存 | ≥2GB |
| 磁盘 | ≥500MB |

### 2.2 获取 Tushare Token

**Tushare 是免费的数据源，需要注册获取 Token：**

1. 访问 [Tushare官网](https://tushare.pro/)
2. 注册账号（免费）
3. 登录后，进入「个人中心」→「接口Token」
4. 复制您的 Token（类似：`2a92206aa63f787d...`）

> **注意**: 免费版 Tushare 每日限制 8000 次调用，足够个人使用。

## 3. 安装配置

### 3.1 克隆项目

```bash
# 克隆代码仓库
git clone <repository-url>
cd stock-analyzer
```

### 3.2 创建虚拟环境

```bash

python3 -m venv .venv

# 激活虚拟环境
# Linux/macOS:
source .venv/bin/activate

# Windows:
# .venv\Scripts\activate
```

### 3.3 安装依赖

```bash
# 安装项目依赖
pip install -r requirements.txt

# 安装开发依赖（可选）
pip install -r requirements-dev.txt
```

### 3.4 配置环境变量

```bash
# 复制配置模板
cp .env.example .env

# 编辑配置文件
nano .env  # 或使用您喜欢的编辑器
```

**`.env` 文件内容示例：**

```ini
# 应用配置
APP_NAME=Stock Analyzer
APP_ENV=development
APP_DEBUG=true

# Tushare 配置（必填）
TUSHARE_TOKEN=your_tushare_token_here

# 数据库配置（可选，暂未使用）
DATABASE_URL=postgresql://user:pass@localhost:5432/stock_analyzer

# Redis 配置（可选，暂未使用）
REDIS_URL=redis://localhost:6379/0
```

**重要**: 请将 `your_tushare_token_here` 替换为您在步骤 2.2 获取的真实 Token。

## 6. 输出说明

### 6.1 报告文件

**输出位置**: `reports/real/` 目录

**文件命名**: `{股票代码}_{日期}.html`

**示例**:
```
reports/real/
├── 688981.SH_2026-04-06.html  # 中芯国际报告
├── 000001.SZ_2026-04-06.html  # 平安银行报告
└── 600519.SH_2026-04-06.html  # 贵州茅台报告
```

### 6.2 报告内容

HTML 报告包含以下信息：

#### 基本信息
- 股票代码 / 名称
- 分析时间
- 报告版本

#### 评分体系
- 综合评分 (1-5)
- 信号强度
- 安全等级
- 评级 (A/B/C/D)

#### 投资建议
- 建议（买入/持有/卖出）
- 置信度 (0-100%)

#### 价格参考
- 建议入场价
- 止损价
- 目标价
- 预期收益率
- 支撑位 / 压力位

#### 风险评估
- VaR(95%)
- 最大回撤
- 风险等级

### 6.3 技术指标说明

| 指标 | 说明 | 参考值 |
|------|------|--------|
| **EMA(20)** | 20日指数移动平均线 | 价格在均线上方为多头趋势 |
| **RSI(14)** | 14日相对强弱指标 | <30超卖，>70超买 |
| **MACD** | 异同移动平均线 | DIF上穿DEA为金叉（买入信号） |
| **ATR(14)** | 14日平均真实波幅 | 反映波动性，越大波动越剧烈 |

## 7. 常见问题

### Q1: 提示 "ModuleNotFoundError: No module named 'app'"

**原因**: 未正确设置 Python 路径

**解决方案**:
```bash
# 确保在项目根目录运行
cd /path/to/stock-analyzer

# 或添加路径
export PYTHONPATH="${PYTHONPATH}:$(pwd)"
```

### Q2: 提示 "Tushare Pro API 初始化失败"

**原因**: Token 未配置或无效

**解决方案**:
```bash
# 检查 .env 文件
cat .env | grep TUSHARE_TOKEN

# 确保格式正确
TUSHARE_TOKEN=2a92206aa63f787d...  # 无引号
```

### Q3: 提示 "获取数据失败或数据不足"

**原因**: 
- Token 积分不足（需要 120+ 积分）
- 股票代码错误
- 网络问题

**解决方案**:
```bash
# 检查 Tushare 积分
# 登录 https://tushare.pro/ 查看积分

# 确认股票代码格式
# 上海交易所: 688981.SH
# 深圳交易所: 000001.SZ
```

### Q4: 报告显示价格与实际不符

**说明**: 当前报告使用的是**不复权数据**

**验证方法**:
```bash
# 测试 Tushare 数据
python3 -c "
import tushare as ts
ts.set_token('your_token')
pro = ts.pro_api()
df = pro.daily(ts_code='688981.SH', start_date='20260403', end_date='20260403')
print(df[['trade_date', 'close']])
"
```

### Q5: 如何分析多只股票？

**方法一**: 修改脚本循环调用
```python
stock_list = ["688981.SH", "600519.SH", "000001.SZ"]
for stock in stock_list:
    asyncio.run(analyze_stock(stock))
```

**方法二**: 使用批量分析 API（即将支持）

### Q6: 测试覆盖率未达到 80%

**说明**: 运行单个测试时覆盖率会偏低

**解决方案**:
```bash
# 运行完整测试套件
python3 -m pytest tests/ --cov=app --cov-fail-under=80
```
